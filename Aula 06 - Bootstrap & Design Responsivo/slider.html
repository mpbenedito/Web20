Slider Simples

Necessário dos recursos do javascript, jquery e popper, além do bootstrap

Classes necessárias: carousel e slide para iniciar. Configurar o atributo data-ride para carousel

Necessário ter a classe carousel-inner para identificar a área do carrossel

Os elementos que vão ser apresentados devem ser classificados como crousel-item. O primeiro é o active

Slider com controles

Abaixo do inner, porém dentro do carrossel são definidos os controles no formato de a href. Eles trazem o ícone de previous e next para dentro da imagem.

Coloquei um span de um ícone pré-definido, porém pode ser um botão ou um ícone do "Iconic" ou "Font Ausesome"

Slider com Indicações

Os controles ficam abaixo da inner e os indicadores acima da inner

Classe utilizada é a carousel-indicators para o container dos indicadores

Os itens dos indicadores precisam utilizar os atributos data-target e data-slide-to

Slider com Legendas

Os textos serão aplicados em containers junto de cada imagem

Classe utilizada é a carousel-caption
