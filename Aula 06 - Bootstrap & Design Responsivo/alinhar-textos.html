Sufixos de tamanho de dispositivos

XL - Extra Large: >= 1200px

LG - Large: >= 992px

MD - Medium: >= 768px

SM - Small: >= 576px

Estes sufixos são utilizados para determinas as classes que podem ser utilizadas para um tipo de dispositivo ou outro, trazendo a responsividade para a página.

Classe: é definida por uma combinação de palavras: text-tela-alinhamento onde:

tela: deve ser substituído pelo sufixo do tamanho que se deseja: sm, md, lg, xl

alinhamento: deve ser substituído pelo alinhamento em tela: left, right, center

Exemplos:

tela >= 576px: text-sm-right

tela >= 768px: text-md-right

tela >= 992px: text-lg-right

tela >= 1200px: text-xg-right

Elementos InlineBlock

Classe d-inlineblock transforma os elementos para terem um comportamento inlineblock.

Relembrando:

Block: o elemento avalia seu conteúdo e para apresentá-lo ocupa um "bloco" da tela disponível, portanto nele é possível aplicar margens ou definir tamanhos, por exemplo.

Inline: o elemento avalia seu conteúdo e ocupa o espaço estritamente necessário para apresentá-lo.

InlineBlock: o elemento ocupa o espaço restrito do seu conteúdo porém aceita que seja alocado em espaços cuja dimensão pode ser definida.

Elementos inline com tamanhos e margens definidas dentro de uma div (laranja largura=500px altura=150px). A div ocupa a área definida porém os dois elementos não precisam ocupar mais do que uma linha. A aplicação de margens verticais não surte efeito, apenas as laterais.

Elementos block com tamanhos e margens definidas dentro de uma div (laranja largura=500px altura=150px). A div ocupa a área definida e seus elementos ocuparam cada um uma linha e a aplicação de margens verticais surte efeito.
