List-group

Classe list-group para formatação de listas, aplicada ao elemento ul, preparando o objeto para as próximas formatações

Classe list-group-item nos elemento li para fazer o padding, criar as caixas arredondadas.

Elementos classificados como list-group ajustam-se automaticamente ao tamanho da tela.

É possível aplicar as classes active e disabled para marcar de forma diferenciada um elemento da lista. No exemplo o item 3 está ativo e o 2 desabilitado.

Badges

Classe: badge

Badges são "marcadores" visuais adicionados aos elementos da lista. Os itens 3 e 4 tem uma badges associadas.

As badges podem ser formatadas utilizando os contextos de cores: badge-opcao, onde opção são os conhecidos contextos primary, secondary, warning, info, etc.
