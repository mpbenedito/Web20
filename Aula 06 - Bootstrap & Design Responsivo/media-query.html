Em CSS puro existem 2 características importantes: Tipo de midia e Resolução

Tipos de Midia

Posso criar estilizações específicas para midias. Exemplo, posso criar um estilo css específico para impressão.

  • all - todos os dispositivos
  • aural - sintetizadores de voz
  • braille - leitores de Braille
  • embossed - impressoras de Braille
  • handheld - dispositivos de mão. P.e. celulares com telas pequenas
  • print - impressoras convencionais
  • projection - apresentação de slides
  • screen - monitores coloridos
  • tty - teleimpressores e terminais
  • tv - televisores

Resolução de telas:

  • 320px - smartphones em modo retrato
  • 480px - smartphones em modo paisagem
  • 600px - tablets pequenos. P.e. Kindle (600 x 800 px)
  • 768px - tablets maiores em modo retrato. P.e. iPad (768 x 1024 px)
  • 1024px - tablets maiores em modo paisagem.
  • 1200px - monitores wide

Em Bootstrap são utilizados Media Queries.

Recurso: @media ( dispositivo-resolução ) { classes de formatação } - definição da midia de apresentação dos conteúdos e a formatação desejada

Os pontos de resolução em que os leiautes se modificam são chamado breakpoints

Conhecendo os breakpoints é possível alterar cores, formas, alinhamentos, margens, posicionamentos.

Lembrar que não é possível utilizar ao mesmo tempo max-width e max-width sozinhos. Os breakpoints devem ser colocados entre intervalos com a clásula "and", num formato mais usual.
