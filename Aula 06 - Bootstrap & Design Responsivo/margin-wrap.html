Wrap (flex-wrap, flex-nowrape)

Por padrão, um container flex obriga os itens a se "expremerem" para que todos caibam dentro de si.

Porém é possível aplicar as classes wrap e nowrap para controlar quando os objetos deverão "pular linha". No-Wrap é o padrão
