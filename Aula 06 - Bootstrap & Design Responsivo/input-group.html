Input Groups

Grupos criados para juntar os elementos utilizados no input de um formulários.

Existem classificações específicas para tratamento de alguns grupos comuns deixando os elementos preparados para receber determinados tipos de informação.

Classe padrão: input-group

Grupo prepend

Prepend são grupos formatados antes do elemento input. Classe: input-group-prepend

O sinal @ está classificado como input-group-text. Utilizei o @, contudo pode ser qualquer string.

@
Grupo append

Append são os grupos formatados após o input. Classe: input-group-append

A classe input-group-text continua sendo utilizada para a parte fixa do elemento input.
