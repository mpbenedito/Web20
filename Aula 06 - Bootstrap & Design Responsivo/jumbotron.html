Alinhamentos: align-self-ALINHAMENTO (start, center, end)

Existem alinhamentos específicos para posicionamento das imagens com os textos. Classes align-self-[posição], onde [posição] pode ser start, center, end. Padrão é start

Reparar que o alinhamento é vertical. Start é no topo do container, center é ao centro e end é embaixo.
