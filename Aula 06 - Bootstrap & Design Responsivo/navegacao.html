Navegação simples

Classe: nav, para menus de navegação. A classe nav aplicada ao elemento list irá subtrair os espaços e colocar os itens juntos numa mesma linha.

Os itens devem ser classificados como nav-item, preparando-os para receberem atributos.

Os elementos "a href" devem ser classificados como nav-link.

A indicação de qual página está ativa é dada pela classe nav-pills diretamente no container da lista e active no item que deve ser destacado.

Da mesma forma que existe o active é possível definir os links como disabled.

Navegação em abas

Manter a classificação nav para o container, nav-item para os itens e nav-link para as referências.

Porém para a criação de abas o container deve ser do tipo nav-tabs. E o active define a aba que deve ser a selecionada no momento.

Barra de navegação simples

Utilizado para criação simplificada de barras de navegação.

Classes: navbar - prepara a criação da barra de navegação. É utilizada no container da barra de navegações (p.e. um elemento div ou nav).

Na definição da lista deve-se utilizar a classe navbar-nav.

Por último, em cada elemento deve-se aplicar a classe

Só com estes elementos já é possível definir a barra, porém ela ficará na vertical.

Será necessário configurar ela na horizontal com a classe navbar-expand-sm/md/lg/xl, que definirá o tamanho de apresentação da barra.

Para criar uma identificação diretente para o logo, de forma que ele não se confunda com o menu de opções, aplica-se a classe navbar-brand nele.

Classes navbar-light e navbar-dark identificam as cores da barra de navegação: clara ou escura.

Definida a cor da barra de navegações define-se o background em que a barra aparecerá (p.e. bg-dark,bg-primary, bg-sedondary, bg-success, etc)
