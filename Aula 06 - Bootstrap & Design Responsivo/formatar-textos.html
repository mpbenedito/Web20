Formatações de texto: normal, negrito, itálico

texto normal: classe font-weight-normal (a criação de uma class deste tipo é importante para utilização em textos majoritariamente em negritos podendo sofrer variação para normal em pequenos trechos. É uma reversão interessante)

texto negrito: classe font-weight-bold

texto itálico: classe font-italic

Truncar textos

Classe: text-truncate - Utilizado para fazer a apresentação de textos longos em telas de tamanho reduzido. Brincar mudando a largura do browser. No exemplo a baixo o texto truncado apresenta ... no final da linha e vai sendo apresentado à medida que o navegador é expandido.
