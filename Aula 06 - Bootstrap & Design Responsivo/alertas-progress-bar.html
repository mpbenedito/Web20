Barra de progresso

Modo básico:

Barras de progresso são elementos para interação visual com usuário informando sobre o andamento de algum tipo de processamento ou carga, p.e.

Classe: progress - colocada no container externo, formata uma região da tela para conter uma barra de incremento

Classe: progress-bar - colocada no container interno, indica a progressão da barra
