Classes de Largura

Para leiautes líquidos é necessário trabalhar com porcentagem nas dimensões dos componentes, então deve-se utilizar as classes específicas.

Classes: w-[porcentagem], onde porcentagem é o número representativo do percentual que o elemento deve ocupar, podendo ser: 25, 50, 75, 100 ou auto.

Exemplos:

w-25: width 25%
w-50: width 50%
w-75: width 75%
w-100: width 100%
w-auto: width auto

Classes de Altura

Utilizadas para determinar a altura de elementos com leiaute líquido

Classes: h-[porcentagem], de forma análoga ás classes de largura, define o percentual de altura.

Exemplos:

h-25: height 25%
h-50: height 50%
h-75: height 75%
h-100: height 100%
h-auto: height auto

Classes de Bordas

Colocação de bordas nos elementos

Classes: border-[direção], onde [direção] define a colocação da borda, podendo ser top, right, bottom, left ou deixada em branco para bordas em todas as direções.

Colocação de cores das bordas

Classes: border-[cor], onde [cor] pode ser primary, secondary, success, info, warning, danger, light, dark, white

Exemplos:

border border-danger: todas as bordas vermelhas
border-top border-success: borda superior verde
border-bottom border-warning: borda inferior laranja
border-right border-info: borda direita azul
border-left border-dark: borda esquerda preta


Bordas Arredondadas

Arredondamento de bodas nos elementos

Classes: rounded ou rounded-[direção], sendo a primeira a classe que arredonda todas as bordas e rounded-[direção] arredondará as bordas da direção especificada.

[direção] pode ser: top, bottom, right, left ou circle, sendo esta a última para a criação de elementos circulares

IMPORTANTE: As classes rounded DEVEM ser precedidas da classe border.
