Margens

Classes: m/mt/mb/ml/mr/mx/my-[número], onde:

m: margem para todos os lados

mt: equivalente a margin-top

mb: equivalente a margin-bottom

ml: equivalente a margin-left

mr: equivalente a margin-right

mx: equivalente a margens horizontais (margin-right e margin-left)

my: equivalente a margens verticais (margin-top e margin-botttom)

[número]: um número entre 0 e 5 equivalente ao espaçamento da margem. A unidade de medida é a do bootstrap (REM). Não são pixels nem "em".

Exemplos: m-0, mt-1, ml-5

Exemplo de aplicação prática: a partir de um parágrafo em laranja, inicio as aplicações de parágrafos em sequência, cujas margens topo estão definidas em sequência de 0 a 5 e as margens inferiores sempre estão zeradas, para que uma não interfira na outra.

Elemento anterior com margens = 0

Conteúdo principal com margem 0 (p-0)

Conteúdo principal com margem 1 (p-1)

Conteúdo principal com margem 2 (p-2)

Conteúdo principal com margem 3 (p-3)

Conteúdo principal com margem 4 (p-4)

Conteúdo principal com margem 5 (p-5)

Elemento posterior com margens = 0

Notar que as margens deslocam os elementos dentro de seu container. No caso todos os elementos parágrafos <p> criaram uma área ao redor de si, porém não mudaram de tamanho.

Margens se sobrepoem. Perceber que as margens de um elemento com m-2 não se somou com os elementos m-2 e m3. Sempre prevaleceu a maior das margens, pois a menor foi montada e a seguinte apenas se sobrepôs.

Padding

Classes: p/pt/pb/pl/pr/px/py-[número], onde:

p: padding para todos os lados

pt: equivalente a padding-top

pb: equivalente a padding-bottom

pl: equivalente a padding-left

pr: equivalente a padding-right

px: equivalente a paddings horizontais (padding-right e padding-left)

py: equivalente a paddings verticais (padding-top e padding-botttom)

[número]: um número entre 0 e 5 equivalente ao espaçamento do padding. A unidade de medida é a do bootstrap (REM). Não são pixels nem "em".

Exemplos: p-0, pt-1, pl-5

Exemplo de aplicação prática: o mesmo elemento é apresentado repetidamente, sempre variando seu padding. O elemento é um parágrafo, sem alteração de suas margens. Apenas o padding é alterado.

Conteúdo principal com padding 0 (p-0)

Conteúdo principal com padding 1 (p-1)

Conteúdo principal com padding 2 (p-2)

Conteúdo principal com padding 3 (p-3)

Conteúdo principal com padding 4 (p-4)

Conteúdo principal com padding 5 (p-5)

Notar que as margens do componente não se deslocam. No caso as margens padrão dos parágrafos são mantidas e aparecem entre os elementos. Porém o elemento muda seu tamanho, conforme o padding cresce.

Também deve-ser perceber que os paddings não se sobrepoem. Eles se somam ás margens e aos próprios paddings anteriores.
