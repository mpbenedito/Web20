Elementos Flutuantes

Classes: float-left e float-right - fazem os elementos "flutuarem" no espaço de trabalho nas respectivas direções informadas.

Notar que da mesma forma que a estilização, as classes tiram os elementos do fluxo normal e sobrepondo-se a outros elementos.

Float Responsivo

Floats aplicáveis ao comportamento responsivo de um site, que só irá considerar a flutuação dependendo da área útil da tela do dispositivo em que está sendo apresentado.

Classes: float-tela-direção. Onde:

tela: é o sufixo que identifica o tamanho de tela trabalhado: sm, md, lg, xl

direção: direção da flutuação que pode ser apenas left ou right

float-sm-right: telas >= 576px
float-md-right: tela >= 768px
float-lt-right: tela >= 992px
float-xl-right: tela >= 1200px

Observar que os elementos de flutuação responsiva também alteram o fluxo de texto da página, portanto é conveniente encapsulá-los num parent cuja classe seja clearfix. Se este pai não existisse, o posicionamento deste texto estaria sendo afetado pelos boxes coloridos acima. Eles só flutuam quando a tela aumenta. Enquanto a tela está menor que a mínima identificada pelo sufixo, eles permanecem sem flutuação.

Elementos colados

Classe: sticky-direção, onde:direção: pode ser top ou bottom

Quando aplicada, esta classe, coloca o elemento no fluxo normal de texto, porém quando a tela é rolada, antes que o elemento desapareça da tela ele é colado ao final da direção indicada.

Importante:O elemento identificado com a classe sticky permanecerá fixo enquanto seu container estiver em tela. Isto significa que se for aplicada numa div dentro de outros elementos, esta div só ficará fixa, enquanto seu container estiver em tela. Para que a fixação seja permanente deve-se utilizar esta classe num container cujo o pai seja o próprio body. O exemplo de lista abaixo foi configurado desta forma.
