Utilização de elementos Flexbox

A aplicação das classes container e row, junto com as classes de posicionamento (col, align, justify, etc), basicamente é a aplicação do conceito de flexbox com a definição do grid de 12 espaços.

Porém é possível aplicar o próprio flexbox a um container sem o row, fugindo do conceito do grid. Os exercícios abaixo simulam esta aplicação, criando containers (cinza) com elementos internos que são modificados, dependendo da aplicação que se está demonstrando.

Flex container e flex itens (d-flex, flex-row, flex-column, flex-row-reverse)

A utilização da classe row cria um flex-container, onde os elementos são flex, porém arranjados em um grid.

Perceber que um elemento d-flex terá toda a extensão do elemento container. A presença dos gutters (espaçamentos) é perceptível, por causa da classe container (em verde)

Sem o row os elementos seriam apenas empilhados, pois o grid não existe, então não há distribuição colunada
