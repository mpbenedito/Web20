Tabelas: table

Novas tags do HTML5 thead, tbody e tfoot, servem para agrupar os elementos do cabeçalho e do detalhamento da tabela. Com elas os navegadores qual parte da tabela participa do scroll de dados, mantendo o cabeçalho fixo no topo, ou numa impressão, mantendo o thead impresso em cada página.

Classe: table - já recebe toda a formatação para o header e para o body (altura, larguras, espaçamentos)

Tabela com formatações: table-striped, table-bordered, table-borderless

table-striped: faz a diferenciação de cores entre linhas pares e ímpares, colocando uma formatação mais escura nas pares

table-bordered: faz a marcação de todas as bordas. Variação: table-borderless para não ter nenhuma borda

Paginação: pagination, page-item, page-link - active, disabled

Neste exercício será feita apenas a estilização dos componentes da paginação, pois para que a paginação realmente ocorresse seriam necessários conhecimentos para páginas dinâmicas, recuperação de informações do banco, os links deveriam procesar as cargas, etc..

Classe pagination aplicada a um elemento container, como uma lista. Isto formará os elementos deixando-os próximos.

Classe page-item aos elementos listados para a navegação.

Classses active e disabled para marcar a página atual e desabilitar alguma. No exemplo página 1 está ativa e o link da página Anterior está desabilitado, pro isso.

Classe page-link aplicado aos links de navegação

Setas

No lugar de apresentar links escritos é possível apresentar sinais formados por entidades html, contudo a utilização destas setas irá causar problemas aos leitores de tela, úteis para pessoas com deficiência visual. Para contorno desta situação os links não são suprimidos; eles são apenas escondidos com a classe sr-only (screen reader only) para serem apresentados apenas para os leitores de tela.
