Template Strings

É uma feature do ES6. Basicamente, template strings pode ser entendido como um recurso que habilita "janelas" ou "lacunas" no texto que será apresentado e, com a interpretação e execução do script, estas lacunas possam ser preenchidas.

Até o momento toda concatenação de textos ocorre de forma bem simples: utiliza-se o sinal de + e o HTML5 interpreta isso como uma concatençaão. Ex.: "Oi " + cNome + ", tudo bem?". Isto produz um texto, e quando a variável cNome for preenchida, o texto passará a ter algum sentido dentro da lógica do script.

Usando um template string, o mesmo exemplo acima deve ser escrito da seguinte forma: `Oi ${cNome}, tudo bem?`. Da mesma forma, quando cNome for preenchido, o texto será completado e a frase terá um sentido dentro da aplicação.

Partindo do exemplo acima, observar as seguintes quetões:

  • o texto deve ser preenchido dentro de crases ` (não utilizar aspas duplas ou simples);
  • a variável que será a lacuna do template string deve estar dentro da estrugura ${};
  • pular linha no texto do javascript causa uma quebra de linha também no texto apresentado, ou seja, as formatações são replicadas entre o que está entre as crases e o que é apresentado no retorno;

Uma forma de entender os template strings é entender que é utilizá-lo é abrir para o intepretador uma possibilidade de atuação diretamente no texto, desta forma, o texto a ser apresentado pode ser resultado de cálculos ou fórmulas.
