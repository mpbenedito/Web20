Default

A iniciativa de se utilizar valores default em funções é para tornar o fluxo de processamento mais robusto e tolerante a falhas. Neste caso, é necessário lembrar que quando a assinatura da função é construída pode-se construir uma função com um número indefinido de parâmetros, porém para o javascript não existe o concieto de parâmetro obrigatório.

Desta forma, uma função com N parâmetros pode ser chamada sem que todos sejam informados. Neste caso, na definição da função podem ser adotados tratamentos de valores default, assim, uma função com N parãmetros pode ser chamada utilizando-se todos os seus parâmetros ou apenas alguns.

Exemplo1: a função UmaFraseQualquer(), sem nenhum parâmetro, e sem um default associado cria resultados inusitados na tela.

Exemplo2: a função UmaFraseQualquer2(), sem parãmetros e com tratamento default, cria outro resultado completamente diferente.

Exemplo3: a função UmaFraseQualquer2(, 'caçar gatos', ), com apenas alguns parâmetros definidos

Importante: um parãmetro não passado de formaa intencional para uma função deve ser substituído pela palavra reservada undefined.

Arrow Function

É uma notação permitida pelo ECMAScript que possibilita a escrita da função com economia de codificação, ganhando otimização na escrita

Uma função tradicional pode ser definida em uma variável, atribuindo a esta um código a ser executado. A sintaxe padrão já foi estudada

let NomeVar = function( p1, p2, p3) {...} . Exemplo: let quadrado = function(x) { return x*x }

A sintaxe básica de arrow functions utiliza-se intrinicamente do conceito de callback, assum uma variável é definida com um bloco de código a ser executdo.

let NomeVar = (p1, p2, p3) => {...}. Exemplo: let quadrado = (x) => { return x*x }

Se a função a ser criada possui apenas um parâmetro a sintaxe pode ser otimizada, com a omissão dos sinais ( e )

let NomeVar = p1 => {...}. Exemplo: let quadrado = x => { return x*x }

Como última otimização se a função possui apenas uma única instrução, esta também já pode ser informada diretamente no lugar do bloco de código
