Posicionamento relativo (relative): o atributo desloca o objeto de sua posição inicial. O fluxo geral não se altera (observar o parágrafo 3)

O parágrafo 2 é relativo a se posicionar um pouco deslocado de onde deveria se posicionar original, mas não altera o fluxo do texto padrão.

Posicionamento absoluto (absolute): O parágrafo 2 é do tipo absoluto e está definido para ficar sempre no canto direito do container que o suporta.

Objetos com posicionamento absoluto só conseguem se posicionar em containers que não sejam estáticos. Neste caso o div atual é do tipo Relative.

O parágrafo 2 por ser "absoluto" no canto, "sai" do fluxo de texto e ocupa a posição para o qual foi definido
