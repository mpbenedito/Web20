Arrays Básicos

São elementos de programação já verificados em outras linguagens, com possibilidade de manipulação em vários códigos com significância de coleções de elementos.

Array é um tipo de variável. Forma de criação: var <nome_variavel> = Array() ou var <nome_variavel> = []

O tamanho do array é definido dinamicamente e não precisa ser declarado em sua criação.

Os índices começam de 0, ou seja, o primeiro elemento de um array é o de índice 0.

Os índices também podem ser alfanuméricos, desta forma são conhecidos como chaves. Inspecionar esta página para entender.

É possível fazer a atribuição direta dos elementos já na declaração: var <nome_variavel> = Array(elemento 0, elemento1, elemento2, elemento3) ou var <nome_variavel> = [elemento 0, elemento1, elemento2, elemento3]

Ponto de atenção: atributo lenght conta apenas os elementos de um array que estão sequenciais começando do elemento 0. Portanto, utilizar índices alfanuméricos pode trazer problemas na hora de contar os elementos.

Arrays Multidimensionais

Os elementos de um array bidimensional são outros arrays, portanto todas as características do array básico são válidas.

Não existe limite para a multidimensionalidade de um array

Inclusão e Exclusão de Elementos

Existem métodos específicos para inclusão e remoção de elementos.

Inclusão: o array é uma pilha, portanto é possível incluir tanto no topo (elemento0) quanto no fim do array (último elemento). Método utilizado para incluir no fim é o push() e para incluir no topo é o unshift().

Exclusão: para escluir elemento no final do array utiliza-se o método pop() e no início do elemento utliza-se o shift(). Não se passa parâmetros, pois pop e shift buscam o primeiro e o último respectivamente.

Os métodos de inclusão e exclusão são válidos tanto para vetores quanto para arrays multidimensionais

Pesquisa em Arrays

Método: indexof(<conteúdo a ser pesquisado>), retornará o primeiro índice ou chave em que o elemento pesquisado foi encontrado

Se o elemento não for encontrado o retorno será -1

Ordenação de Arrays

Método: .sort() age sobre o próprio array realinhando os elementos e ajusatando os índices. É utilizado especificamente para ordenação alfanumérica.

Ordenação em valores numéricos é necessário trabalhar com função específica de callback. Da mesma forma deve se criar um callback para ordenação decrescente e outros exemplos
