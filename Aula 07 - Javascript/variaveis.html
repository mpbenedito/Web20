Declaração e atribuição

Sintaxe base: var [nome_da_variavel] = [conteúdo]

Para Strings utilizar o conteúdo entre aspas simples ou duplas. Ex: var cTexto = "curso"

Variáveis numéricas podem ser inteiras ou float, positivos ou negativos: var nInteiro = -8 ou var nFracao = 0.8

Variáveis lógicas são as variáveis padrão boolean que podem ser apenas True ou False. Ex.: var lBoolean = false ou var lTeste = true

Não existe definição do tipo da variável. Não se faz nada do tipo "Dim cVariavel as String". O javascript reconhece o tipo da variável pelo conteúdo que está sendo atribuído.

Atribuir 'true' para uma variável é diferente de true. Na primeira é um texto, na segunda é um valor lógico.

Ao abrir as ferramentas de desenvolvedor consigo enxergar a atribuição das variáveis.

Função alert() serve para apresentação de informações em tela, por exemplo recuperar o conteúdo de uma variável.

Método document.write() força a exibição de informações diretamente no corpo da página. Permite o acesso direto ao DOM da página para recuperar ou gravar conteúdos dos scripts

Aplicar escritas diretamente no console log da página, que pode ser consultado diretamente nas ferramentas de desenvolvedor que o navegador oferece.

Concatenar strings também é possível, como em outras linguagens. O operador de concatenção é o +. Quando concatenar um número a um texto, o javascript faz o parser automaticamente.

Ponto de atenção: variáveis podem ser nativamente null ou undefined. Uma variável definida null indica que intencionalmente ela não tem valor atribuído. Uma variável undefined significa que ainda não tem atribuição alguma. São valores diferentes. Então sempre testar se a variável é null e undefined.
