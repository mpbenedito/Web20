Eventos

No javascript é possível capturar os eventos de vários elementos que estão em tela ou momentos em que ações acontecem.

Eventos do Mouse

Os 5 principais eventos que podem ser capturados do mouse: onclick, ondblclick, onmouseup, onmouseover, onmouseout.

Os eventos são configurados como atributos do elemento com o qual se deseja interagir.

Faça a interação com as caixas abaixo e acompanhe o código desta página.

OnClick
OnDblClick
onMouseUp
onMouseOver
onMouseOut

Nem todos os eventos de mouse estão disponíveis para todos os elementos do javascript. É necessário consultar documentação para saber quais estão disponíveis.

Eventos do Teclado

Análogos aos eventos de mouse é possível avaliar codificar ações para eventos que acontecem no teclado.

Os principais eventos de teclado são: onkeydown, onkeypress e onkeyup.

Eventos de Formulário

São eventos disparados quando da interação com elementos de um formulário.

Os principais eventos de teclado são: onfocus, onblur e onchange.

Eventos de Janela (frame)

São eventos disparados quando da interação com a janela de exibição da página e algum elemento sofre redimiensionamento.

Os principais eventos de teclado são: onresize e onscroll.

Alterar o tamanho da tela e fazer as rolagens. Geralmente o elemento que recebe a parametrização é o body.
