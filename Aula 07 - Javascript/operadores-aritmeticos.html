Operação Aritmética na Atribuição de Variáveis

É possível realizar as operações aritméticas das variáveis diretamente em sua atrição. Exemplo.

A atribuição "nVar += 3" é equivalente a "nvar = nvar + 3"

A atribuição "nVar /= 4" é equivalente a "nvar = nvar / 4"

Esta operação também é viável para concatenação de strings.

Precedência

Os princípios de matemática básica são utilizados.

As equações são resolvidas na ordem em que aparecem da esquerda para direita

Primeiro são resolvidas as multiplicações e divisões na ordem em que aparecem.

Em seguida são resolvidas as somas e subtrações.

Exemplo: ( 11 + 5 - 5 * 2 / 2 ) = 11

Os parênteses podem alterar a sequência de cálculo. O javascript resolve os parênteses mais internos, utilizando as regras já conhecidas, seguindo para os externos. A utilização de parênteses altera a precedência dos cálculoa

Exemplo: ( ( 11 + 5 - 5 * 2 ) / 2 ) = 3
