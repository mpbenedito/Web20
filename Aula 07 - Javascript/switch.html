Switch..Case

Com base em um determinado valor o Switch..Case também é um comando condicional pelo qual pode-se seguir diversos fluxos no código.

Switch sempre trabalha de forma implícita com o operador Igual e Idêntico (===)

switch (variável)

 case valor1

   [Relação de Comandos }

  break

 case valor2

  { Relação de Comandos }

  break

 default

  { Relação de Comandos }

  break

Se o comando break não for utilizado o fluxo de execução seguirá interpretando e executando os comandos dos cases abaixo. O break interrompe o fluxo de execução e o coloca logo abaixo do último break
