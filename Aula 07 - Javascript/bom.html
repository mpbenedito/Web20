BOM - Browser Object Model

É uma API que viabiliza a interface do javascript com o browser, por exemplo, acessar o histórico de navegação, salvar e recuperar cookies, redirecionar uma página. Entenda-se que não se está acessando recursos de objetos html e sim recursos do browser em que a página está sendo executada.

O BOM possui alguns componentes que podem ser acessados nativamente pelo javascript, pois estes componentes são criados como elementos acessíveis ao fluxo de execução.

Se o DOM é a árvore que relaciona os elementos que foram criados na execução das instruções da página, o BOM é o conjunto de recursos que podem ser acessados e manipulados para que estes objetos sejam exibidos.

Objeto Window

O objeto Window é uma representação da janela do browser onde todo o conteúdo será renderizado. O objeto Window é formado pelo browser e incorpora o DOM, ou seja o objeto "document", já visto, é um componente do Window.
