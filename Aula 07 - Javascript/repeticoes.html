Estruturas de Repetição

Estruturas de execução de código de forma repetitiva, porém com alterações dinâmicas possibilitando a redução de código.

While

Sintaxe básica: while (<condição>) {...}

Enquanto a <condição> for verdadeira o laço será executado. Se a condição se inicia em falso a repetição nem ocorre.

A instrução break causa a interrupção da estrutura de repetição e desvia o fluxo de interpretação para as instruções seguintes ao bloco While().

A instrução continue também causa a interrupção do bloco de comandos e força uma nova avaliação da condição while, e o fluxo de interpretação dos comandos continua do início do while()

Do..While

Sintaxe básica: do {...} while (<condição>)

A avaliação da condição só ocorre após a primeira execução do bloco de códigos. E enquanto a condição for verdadeira o bloco será repetido. Quando a condição tornar-se falsa o bloco de código para de ser repetido.

As instruções break e continue também são válidas nas estruturas do..while e funcionam de forma análoga.

For

Sintaxe básica: for( variável; condição; controle) {...}

A variável utilizada não precisa ser declarada fora da estrutura de repetição

A estrutura for resume as avaliações de repetições baseadas em valores numéricos.

Se a condição se inicia de forma falsa o bloco de códigos nem é executado.

As instruções continue e break funcionam no for.

For..in

Sintaxe básica: for ( variável in array ) {...}

Forma prática de se trabalhar com arrays, o For..in executa o bloco de códigos para cada elemento de um array que for indicado.

A variável não recebe o elemento, mas o índice do array. Assim que o bloco de código é executado o índice do array se incrementa automaticamente.

As instruções continue e break funcionam no for..in.

Todas as posições do array são recuperadas até mesmo aquelas que estão indexadas por uma chave alfanumérica.

Outra funcionalidade é que o for..in percorre todas as posições do array que são válidas. As posições vazias naõ são listadas.

ForEach

Por definição do javascript o foreach não é uma estrutura de repetição, pois é uma função que trabalha com repetição, portanto pode ser avaliado no contexto destes exercícios. Trabalha de forma muito parecida com o for..in

Sintaxe Básica: <variável_array>.foreach( _função_callback( valor, índice, variável_array ) )

Por se tratar de um método do prototype Array(), o foreach() encaminha para a função de callback 3 parâmetros: valor do elemento, o índice dele e o próprio array.

Ao ser chamado, o método foreach executará o conteúdo da função de callback para cada elemento do array, porém elementos indexados por chave alfanuméricas são ignorados, ou seja, o foreach só funcionará em elementos com índice numérico. Elementos em branco não são considerados.
