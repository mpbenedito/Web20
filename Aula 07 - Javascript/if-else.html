If - Else

Controle de fluxos condicional: if (condição) {...} [else {...}]

Variação para o fluxo condicional é o encadeamento, do tipo if (condição) {...} else if (condição2) {...} else {...}

As condição são conseguidas pelos operadores de comparação:

Igual == se os valores comparados são iguais

Idêntico === se os valores comparados são iguais e do mesmo tipo

Diferente != se os valores comparados são diferentes

Não Idêntico !== se os valores comparados são diferentes e de tipos diferentes

Menor < verifica se o valor da esquerda é menor que o da direita

Menor > verifica se o valor da esquerda é maior que o da direita

Menor ou igual <= verifica se o valor da esquerda é menor ou igual que o da direita

Maior ou igual >= verifica se o valor da esquerda é maior ou igual que o da direita

Ponto de atenção: '2' é igual a 2 (comparar somente valor), porém '2' não é idêntico a 2 (comparar valor e tipo). Então fazer atenção ao uso de == ou === nas comparações. O javascript faz um cast implícito de valores.

Casting

Conversão explícita de tipos das variáveis

tostring(): transforma números em texto. tostring não é função. É um método aplicao á classe var. Exemplo: nNotaAluno.tostring() é igual '7'

parseInt(): converte textos em números. Exemplo parseint('2') é igual a 2

parseint('7.5') retorna um resultado de 7

parsefloat(): também converte o texto em número, porém preserva a parte fracionária.

parsefloat('7.5') retorna um resultado de 7.5

Operadore Lógicos

Operador "E" && possibilita o teste condicional (condição1) E (condição2). Se ambas forem verdadeiras o teste é válido.

Operador "Ou" !! possibilita o teste condicional (condição1) OU (condição2). Se apenas uma das condições for verdadeira o teste é considerado válido.

Operador "Não" ! possibilita o teste inverso, ou seja, NÃO (condição). Se a condição for falsa o teste será válido.

Operador Ternario

Forma condicional enxuta que permite teste de condições e possibilita apenas á atribuição de valores.

Sintaxe: (condição) ? <ação_verdadeiro> : <ação_falso>

Exemplo: var cResultado = (nNota >= nMedia && nFaltas <= nMaximoFaltas ) ? 'Aprovado' : 'Reprovado'
