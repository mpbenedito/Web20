Normalize CSS
Cadastro Nome:
Senha:

Dependendo do navegador existem diferenças na exibição de elementos.

O normlize.css é um tratamento feita via estilos que tenta criar um padrão entre os elementos compatibilizando tamanhos, formas, distâncias do HTML em diversos navegadores

Exemplo: carregar esta página em diversos navegadores, sem o normlize.css e depois com. Haverá diferenças significativas neste form simples sem o normalize.css. Com ele, as diferenças serão suavizadas.
