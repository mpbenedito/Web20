Flexbox é uma implementação do CSS3 para facilitar a colocação dos elementos nos containers definidos no projeto.

Os exemplos apresentados aqui mexem apenas nos containers, sem posicionar os conteúdos numerados. Desta forma, é possível alguns leiautes já trabalhando o espalhamento em tela, sem se preocupar com as divisões.

flex-direction: os elementos na codificação não mudam de posição. Apenas a definição do flex-direction que diz como serão apresentados.

Exemplos de utilização: row, row-reverse, column, column-reverse.
