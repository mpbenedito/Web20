Trabalhando com tamanhos de fontes aplicados aos objetos conteiners

O conteiner "div" está definido para apresentação de textos com fonte 40px. Então 40px será o tamanho padrão do conteiner div

Esta linha apresenta o texto com 2em, ou seja 2x o tamanho do fonte do objeto conteinar (div=40px), portanto este texto está escrito com um fonte de 80px.

Esta linha apresenta o texto com 0.5em, ou seja metade do tamanho do fonte do objeto conteinar (div=40px), portanto este texto está escrito com um fonte de 20px.
