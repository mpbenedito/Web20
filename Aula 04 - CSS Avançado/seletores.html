Seletores descendentes são utilizados para formatar elementos que estejam aninhados em outros, independentes de serem filhos diretos.

Este paràgrafo tem um LINK dentro de uma classe. Apenas o link possui uma formatação específica, mesmo não sendo filho da div.

Seletor Irmão Adjacente (este título é um h3)

Este paràgrafo é considerado irmão do título anterior pois ambos são filhos da div em que estão alinhados. E são adjacentes pois são consecutivos, portanto, ele recebe a formatação específica.

Em um contexto geral, o "irmão adjacente" é o elemento consecutivo ao anterior.

Este parágrafo também é considerado irmão do anterior e do título h3, porém não adjacente a esse e não receberá a formatação, seguindo o padrão já definido.

Seletor Irmão Geral (este título é um h3)

Esta formatação aplica-se aos elementos definidos que sejam irmãos em hierarquia, desde que sejam adjacentes entre si, mesmo não sendo consecutivos.
