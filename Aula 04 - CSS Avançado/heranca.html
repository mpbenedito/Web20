Herança - Todo o corpo (body) está definido como vermelho

No body é importante formatar propriedades que servirão para o documento todo: família de fontes, tamanho, cores do fonte

Porém os atributos border e background dos elementos não herdados. Senão todo componente teria uma borda e o background definido do body.

Se fosse uma configuração que deveria servir para todos os elementos da página aí o seletor padrão seria o mais indicado: * {...}

Especificidade

Parágrafo1 - Um seletor mais específico terá maior precedência sobre os gerais, portanto é o que será utilizado para definição dos atributos do elemento.
